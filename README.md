# Shivani Suryawanshi — Professional AI/ML Portfolio

A modern, high-performance, and responsive 3D developer portfolio website built using React, Vite, TailwindCSS (v4), Framer Motion, and lightweight Three.js assets.

Live Demo: **[https://shivani-suryawanshi-portfolio.vercel.app](https://shivani-suryawanshi-portfolio.vercel.app)**

## Features
- **Modern Design**: Dark-mode primary layout utilizing sleek glassmorphism panels, space-inspired color variables, and professional typography (*Space Grotesk* for headings, *Inter* for body, and *JetBrains Mono* for specs).
- **Lag-Free Interaction**: Interactive custom cursor tracking optimized with requestAnimationFrame (RAF) to prevent state re-render loops.
- **Subtle 3D Background**: High-performance starfield particle system combined with rotating geometric wireframe meshes.
- **Dynamic Projects Showcase**: 3D hover-to-reveal tilt flip cards detailing hands-on AI/ML models and data analysis projects.
- **Comprehensive Sections**: Home, About, Skills (grouped), Projects, Experience, Education, Certifications, GitHub Stats, Testimonials, and a fully functional Contact form.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: TailwindCSS v4 + Vanilla CSS
- **Animations**: Framer Motion
- **3D Graphics**: React Three Fiber (R3F) + Three.js + Drei
- **Smooth Scroll**: Lenis Scroll
- **Icons**: Lucide React
- **Deployment**: Vercel

## Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shiva-18-cs/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the local development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
